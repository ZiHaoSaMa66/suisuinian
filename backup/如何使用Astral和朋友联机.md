如题. 你准备和你的朋友MC联机 
你的朋友给你扔了这么一个分享链接 
`astral://room?code=xxxxxxxxxxxxxxxxxx`
那如何使用这个房间分享链接呢

首先你需要下载`Astral`这个组网软件 
（已下载或朋友丢安装包跳过这一步）
在以下渠道中选择合适的渠道下载

[官方网站 夸克盘下载](https://pan.quark.cn/s/78e414fc150d)

[官方 Github Latest Release windows 安装包 (国内下载加速)](https://gh-proxy.com/https://github.com/ldoubil/astral/releases/latest/download/astral-windows-x64-setup.exe)
[官方 Github Latest Release windows 压缩包 (国内下载加速)](https://gh-proxy.com/https://github.com/ldoubil/astral/releases/latest/download/astral-windows-x64-zip.zip)

[官方 Github Latest Release windows-x64-setup](https://github.com/ldoubil/astral/releases/latest/download/astral-windows-x64-setup.exe)
[官方 Github Latest Release astral-windows-x64-zip](https://github.com/ldoubil/astral/releases/latest/download/astral-windows-x64-zip.zip)

下载完成后安装软件
本文编写时使用的软件版本为 `v2.7.8` 
> 更新版本的界面可能有所不同      
> 仅做大致流程参考      
     
1. 打开软件 点击左侧侧边栏的 `房间按钮`

<img width="1186" height="668" alt="Image" src="https://github.com/user-attachments/assets/ba50b882-d80d-46ac-832b-fd1b6994adae" />
       
2. 点击右下角 从左往右数的第2个按钮（类剪贴板图标）

<img width="1186" height="668" alt="Image" src="https://github.com/user-attachments/assets/ef5ff3c1-15c7-487f-ae24-ddac379f957d" />

3. 如弹窗的文字提示 在上方的文本输入框 `粘贴朋友给你的分享链接`      
后 `点击导入按钮`      
或者点击下方的 `从剪贴板导入`      

<img width="1186" height="668" alt="Image" src="https://github.com/user-attachments/assets/ee644ea9-d1cd-4d02-b8e4-789d38e55cfe" />

4.确保你选中了正确的房间

5.点击左侧侧边栏的 `主页按钮` 回到主页面
接着将`自动`开关打开 
将快捷网络设置的 `防火墙` 开关关闭 
最后点击此页面右下角的`连接` 按钮

<img width="1186" height="668" alt="Image" src="https://github.com/user-attachments/assets/5d48d1a8-ab5e-4d01-a883-a24a41e1a79b" />
       
6. 此时点击侧边栏的`房间`页 查看当前连接的状态 检查是否有服务器节点
和您的朋友

7.如连接没有问题 则回到MC当中输入您朋友的IP地址+端口号就可以愉快的联机了
